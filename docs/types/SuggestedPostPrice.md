# SuggestedPostPrice

Describes the price of a suggested post.

## Fields

| Name     | Type     | Required | Description                                                                                                                                                                                                                                                                   |
| :------- | :------- | :------: | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| currency | `string` |   Yes    | Currency in which the post will be paid. Currently, must be one of “XTR” for Telegram Stars or “TON” for toncoins                                                                                                                                                             |
| amount   | `number` |   Yes    | The amount of the currency that will be paid for the post in the smallest units of the currency, i.e. Telegram Stars or nanotoncoins. Currently, price in Telegram Stars must be between 5 and 100000, and price in nanotoncoins must be between 10000000 and 10000000000000. |

## Event Handlers

You can listen for SuggestedPostPrice events using:

```typescript
// Type-specific handler
bot.onSuggestedPostPrice(async (suggestedpostprice: SuggestedPostPrice) => {
  console.log('Received:', suggestedpostprice);
});

// Generic handler
bot.on('suggestedpostprice', async (data: SuggestedPostPrice) => {
  // Same as above
});
```
