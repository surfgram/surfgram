# refundStarPayment

Refunds a successful payment in Telegram Stars. Returns True on success.


## Parameters

| Parameter | Type | Required | Description |
| :--- | :--- | :---: | :--- |
| `userId` | `number` | Yes | Identifier of the user whose payment will be refunded |
| `telegramPaymentChargeId` | `string` | Yes | Telegram payment identifier |



## API Documentation

For more details, see the official [Telegram Bot API documentation](https://core.telegram.org/bots/api#refundStarPayment).
