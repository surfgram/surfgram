# setMyName

Use this method to change the bot&#39;s name. Returns True on success.


## Parameters

| Parameter | Type | Required | Description |
| :--- | :--- | :---: | :--- |
| `name` | `string` | No | New bot name; 0-64 characters. Pass an empty string to remove the dedicated name for the given language. |
| `languageCode` | `string` | No | A two-letter ISO 639-1 language code. If empty, the name will be shown to all users for whose language there is no dedicated name. |



## API Documentation

For more details, see the official [Telegram Bot API documentation](https://core.telegram.org/bots/api#setMyName).
